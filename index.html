      👋 Welcome to the JO Tracker

      This is a live read-only view of all open Job Orders from Bullhorn. Here's what you can do:

      • 🔍
        Search and filter Type a job title, client, candidate name, or JO ID in the search box. Use the dropdowns to narrow by priority, sales owner, or recruiter.
      • 👁
        Tap any row to see candidates The arrow on the left expands the row and shows everyone active on that job.
      • ⇅
        Sort by any column Click a column header (like "Priority" or "Active") to sort. Click again to reverse.
      • ↻
        Daily updates (Olga only) Olga runs the Amplify report → clicks Load Fresh Report → clicks Publish Update. Everyone sees new data within ~60 seconds. No refresh needed on your end.
      • 🔒
        This is read-only You can't edit job orders here. Any changes (priorities, adding candidates) happen in Bullhorn. This tracker just shows what's there.

      📧 Share this tracker

      Just copy the link below and paste it into a Slack message, email, or Teams chat. Anyone who clicks it will see the latest data — no login required, works on phone and desktop.

      The link never changes. Bookmark it and refresh any time to see updates.

      ↻ Load a Fresh Report

      Run your Amplify report in Bullhorn, then upload the file or paste the content below. After loading, click Publish Update to push the new data to the website.
